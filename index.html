Featured Project

Maastri — Women Safety & Mental Health Support

Full-Stack Web Application

Building safety and support into one platform

Maastri is a web-based application developed to enhance women's safety and promote mental well-being. It combines real-time safety features with an AI-powered chatbot for basic mental health support and a secure personal diary — creating a reliable, user-friendly platform for both protection and emotional care. Built with a responsive front-end and a Flask + SQL Server backend, with secure registration, login, and personalized dashboards.
